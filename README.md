# 🚚 Val Fretes & Mudanças

Landing page moderna e profissional desenvolvida para a empresa **Val Fretes & Mudanças**, com foco em conversão de clientes e apresentação dos serviços de forma clara e elegante.

---

## ✨ Sobre o Projeto

Este projeto é uma landing page responsiva criada para destacar os serviços de mudanças e fretes, transmitindo confiança, organização e profissionalismo.

O design segue um estilo clean e moderno, inspirado em portfólios tecnológicos.

---

## 🛠️ Funcionalidades

* ✅ Layout responsivo (mobile e desktop)
* ✅ Integração com WhatsApp
* ✅ Formulário de contato
* ✅ Seção de serviços
* ✅ Galeria de trabalhos
* ✅ Animações suaves
* ✅ Design moderno e minimalista

---

## 🎨 Tecnologias Utilizadas

* React
* Vite
* Tailwind CSS
* JavaScript / TypeScript
